# 곡괭이 탐지기 (Picks & Shovels Detector)

산업 밸류체인에서 "금 캐는 사람(완성품 우량주)"이 아니라
"곡괭이 파는 사람(필수 공급사)"을 객관적 데이터로 찾는 도구.

## 이게 뭔가
- 주가 예측기가 아님. **구조 분석 + 후보 추리기** 도구.
- 모든 판단은 밸류체인 그래프 + 재무지표에 근거.
- 결과는 "사라"가 아니라 "더 깊이 볼 가치가 있다"는 후보 리스트.

## 구조
```
valuechain/
├── data/source.py        데이터 소스 추상화 (pykrx/토스 갈아끼우기)
├── core/graph.py         밸류체인 그래프 + 충격 전파
├── analysis/
│   ├── shovel.py         곡괭이 점수 (대체불가/고객분산/상류/재무)
│   ├── maturity.py       이미 비싼 곡괭이 vs 숨은 곡괭이 분류
│   └── debate.py         멀티 AI 토론 뼈대 (API 키 필요)
├── demo.py               충격 전파 데모
├── demo_shovel.py        곡괭이 랭킹 데모
└── demo_full.py          전체 파이프라인 데모 ★
```

## 바로 실행
```bash
pip install pykrx finance-datareader networkx
python3 demo_full.py
```

## 실제 사용 3단계

### 1단계 (지금 당장) — 수동 리서치 보조
1. 관심 산업 하나 선택 (예: 로봇, 피지컬 AI)
2. 기업 10~20개의 공급관계·재무를 직접 조사
   - 공급관계: DART 사업보고서의 "주요 매출처/매입처"
   - 재무: 네이버 금융 / 증권사 (PER, 영업이익률, 매출성장)
3. demo.py의 build_xxx_chain() 과 demo_full.py의 fin/sig 딕셔너리에 입력
4. 실행 → 곡괭이 점수 + 4분면 분류 확인
   → 네 직관이 데이터로 맞는지 검증됨

### 2단계 (다음) — 데이터 자동화
- source.py의 PykrxSource 구현 → 시세·시총·PER 자동
- DART 오픈API(무료)로 재무 자동
- 공급관계는 반자동 (공시 추출 + 직접 검수)

### 3단계 (최종) — 멀티 AI 의견
- debate.py에 Claude/Gemini API 붙이기
- 곡괭이 상위 종목을 Bull/Bear 토론에 투입

## 점수 해석
- **곡괭이 점수 높음 + 시장인식 높음** = 이미 검증됨(안전, 상승여력 제한)
- **곡괭이 점수 높음 + 시장인식 낮음** = ★ 숨은 곡괭이 (재평가 여력)
- **곡괭이 점수 낮음** = 완성품/테마주, 또는 관심밖

## 주의
- 투자 자문 아님. 백테스트 전까지 매매 신호로 쓰지 말 것.
- 점수는 "후보 추리기"일 뿐. 최종 판단은 직접.
